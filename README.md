# QuoteApp

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 13.3.1.It is a website that displays quotes, alows users to add their own quotes and upvote or downvote to existing quotes.

## Author

Wanjiru Charity.

## Description

Users can create a new quote.
They can Upvote or downvote a quote.
They can delete a quote
They can see the number of upvotes and number of downvotes for each quote.
